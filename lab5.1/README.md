README

-Open the command line 


-Go to the folder where is located all the downloaded files.
	
	command: cd /..../LOCATION


-Compile all programs using makefile
	
	command: make

-Compile one program using makefile
	
	command: make "FILENAME"


-Run the program
	
	command: ./FILENAME.o

-Clean .o files
	
	command: clean
